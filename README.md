# SENTIMENT ANALYSIS ON GENSHIN IMPACT APP REVIEW
This project conducted sentiment analysis on 1501 Genshin Impact app reviews from Google Play Store. Python code run using Google Colab used to scrapes the reviews, collecting text, rating, and date, which are then preprocessed and normalised. Categorical data is encoded into numerical labels for analysis.


Three machine learning models including Deep Learning, K-Nearest Neighbour, and Decision Tree are built using RapidMiner. The dataset is split into training and testing. For the model performance, this project will evaluate it using accuracy, precision, recall, and detailed classification result will be analysed using confusion matrix.
